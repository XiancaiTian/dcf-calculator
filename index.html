About the DCF Calculator

The DCF Calculator is a web-based tool that helps investors evaluate stocks based on their intrinsic value. It calculates the present value of future cash flows and compares it with the current market price.

Features
  • Calculate DCF value based on projected cash flows
  • Compare current market price with calculated fair value
  • View historical and industry valuation metrics (P/E and P/B ratios)
  • Analyze company's valuation relative to industry peers
  • Project future cash flows with customizable growth rates
  • Quick access to Futu and Yahoo Finance pages for additional research
Technologies Used
  • Backend: Python, Flask
  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Data Source: Yahoo Finance API (yfinance)
